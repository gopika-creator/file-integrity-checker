# File Integrity Checker

## Overview

File Integrity Checker is a Python-based cybersecurity tool that uses SHA-256 hashing to verify file integrity.

The tool generates a unique hash for a file and compares it against a previously stored hash to detect unauthorized modifications or tampering.

## Features

- SHA-256 hash generation
- Save file hashes
- Verify file integrity
- Detect file modifications
- Simple command-line interface

## Technologies Used

- Python 3
- hashlib
- os
